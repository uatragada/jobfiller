# Generate Queue

## Global Prompt Prefix

Create a high-fidelity desktop SaaS application screen inspired by the provided JobFiller reference screenshot. The design should use a clean Untitled UI React style: white and near-white surfaces, subtle gray borders, compact typography, blue primary actions, dense admin-console spacing, fixed left sidebar, global top command bar, central work area, and a right-side inspector panel. The UI should feel like Linear plus a professional ATS operations dashboard. Use small 12-14px text, compact tables, hairline dividers, rounded-xl cards, subtle selected-row blue state, no heavy shadows, no playful colors, no marketing layout, no giant cards. Render at desktop web-app aspect ratio.

## Page-Specific Prompt

Design the Generate Queue page. Use the same JobFiller shell and visual density. Main header says Generate Queue with a count badge and subtitle about generation jobs. Top cards show Queued, Running, Needs Review, and Completed Today. Include a dense queue table with columns Priority, Task, Company, Role, Model, Status, Tokens, Created, Ready, Actions. Select a Stripe cover letter or answer generation task. Right inspector shows generation task details, prompt inputs, model settings, output preview, review notes, and actions Approve, Regenerate, Cancel, Send to Queue. Include a compact Start/Pause Queue control near the header.
